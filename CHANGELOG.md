# 📝 Changelog — Обновление VPN-бота

## Версия 2.0 (27 февраля 2026)

### ✨ Новые функции

#### 1. Админ-панель
- **Админ-меню** (`/admin`) с кнопками
- **Статистика**: пользователи, активные, доход, подписки
- **Управление пользователями**: просмотр списка, информация по ID
- **Рассылки**: отправка сообщений всем активным пользователям
- **Промокоды**: создание/удаление промокодов с лимитами
- **Настройки**: канал, длительность триала

#### 2. Подписка на канал
- **Обязательная подписка** для доступа к функциям
- **Проверка подписки** через Telegram API
- **Кнопка "Я подписался"** с автоматической проверкой
- **Настройка канала** через `.env` или `/set_channel`

#### 3. Бесплатный тариф
- **30 дней бесплатно** за подписку на канал
- **Однократное использование** (защита от повторного)
- **Автоматическая активация** через 3x-ui

#### 4. Реферальная система
- **Реферальные ссылки**: `t.me/bot?start=123456`
- **Бонус за реферала**: +3 дня (настраивается)
- **Статистика рефералов**: список, количество
- **Автоматическое начисление** бонусов

#### 5. Промокоды
- **Создание промокодов** с лимитом использований
- **Бонусные дни** за активацию
- **Защита от повторного** использования
- **Статистика использования**

### 🗄️ Изменения в базе данных

- `promo_codes` — промокоды

- `referrals` — рефералы

### 🚀 Как обновиться

1. **Сделайте бэкап**:
   ```bash
   cp vpn_bot.db vpn_bot_backup.db
   cp -r .env .env.backup
   ```

2. **Обновите файлы**:
   ```bash
   # Скопируйте новые файлы
   # Или сделайте git pull
   ```

3. **Обновите `.env`**:
   ```bash
   # Добавьте новые переменные из .env.example
   nano .env
   ```

4. **Обновите БД**:
   ```bash
   source venv/bin/activate
   python -c "import database as db; db.init_db()"
   ```

5. **Перезапустите бота**:
   ```bash
   sudo systemctl restart vpnbot
   ```

### ⚠️ Breaking Changes

- **Требуется настроить CHANNEL_ID** для работы подписки
- **Старые пользователи** должны переподключиться к каналу
- **Изменены тарифы** — обновите цены при необходимости
