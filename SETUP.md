# 📋 Инструкция по настройке VPN-бота

## 1. Создание канала/группы

### Вариант A: Публичный канал (рекомендуется)

1. В Telegram создайте новый канал:
   - Нажмите "Создать канал" в меню
   - Выберите тип "Публичный канал"
   - Придумайте username (например, `@vpn_yourname`)

2. Добавьте бота в канал как администратора:
   - Зайдите в настройки канала → Администраторы
   - Нажмите "Добавить администратора"
   - Найдите вашего бота по username
   - Дайте права на чтение сообщений

3. Получите ID канала:
   - Добавьте бота [@userinfobot](https://t.me/userinfobot) в канал
   - Он покажет ID (например, `-1001234567890`)
   - Удалите @userinfobot из канала

### Вариант B: Частная группа

1. Создайте частную группу
2. Добавьте бота как участника
3. Получите ID через @userinfobot

## 2. Настройка .env файла

```bash
# Скопируйте пример
cp .env.example .env

# Отредактируйте
nano .env
```

### Обязательные поля:

```ini
# Telegram Bot
BOT_TOKEN=1234567890:AABBccDDeeFFggHHiiJJkkLLmmNNooP
ADMIN_IDS=830960097  # Ваш Telegram ID

# Channel для обязательной подписки
CHANNEL_USERNAME=@your_channel_name     # без пробелов
CHANNEL_ID=-1001234567890               # начинается с -100

# 3x-ui Panel
XUI_URL=http://your-server-ip:54321
XUI_USERNAME=admin
XUI_PASSWORD=your_password
XUI_INBOUND_ID=1
```

### Как получить ADMIN_IDS:
1. Напишите боту [@userinfobot](https://t.me/userinfobot)
2. Он покажет ваш ID (например, `830960097`)

## 4. Проверка работы

### Тест подписки на канал:

1. Запустите бота: `/start`
2. Нажмите "📦 Тарифы"
3. Если не подписаны — бот попросит подписаться
4. Подпишитесь на канал
5. Нажмите "✅ Я подписался"
6. Бот подтвердит и откроет доступ

### Тест бесплатного тарифа:

1. Нажмите "🎁 Бесплатно"
2. Проверьте подписку на канал
3. Нажмите "✅ Активировать бесплатно"
4. Бот выдаст ключ доступа на 30 дней

### Тест админ-панели:

1. Нажмите `/admin` (только для ADMIN_IDS)
2. Выберите "📊 Статистика"
3. Проверьте данные

## 5. Админ-команды

| Команда | Описание |
|---------|----------|
| `/admin` | Открыть админ-меню |
| `/stats` | Показать статистику |
| `/user <ID>` | Информация о пользователе |
| `/promo create CODE DAYS [MAX]` | Создать промокод |
| `/promo delete CODE` | Удалить промокод |
| `/set_channel @username` | Установить канал |
| `/set_trial_days N` | Длительность триала |
| `/grant <ID> <plan> <pay_id>` | Выдать доступ вручную |

## 6. Промокоды

### Создать промокод:
```
/promo create WELCOME30 30 100
```
- `WELCOME30` — код промокода
- `30` — дней бонуса
- `100` — максимум использований (опционально)

### Использовать промокод:
Пользователь нажимает "🎁 Промокод" и вводит код.

## 7. Реферальная система

### Как работает:
1. Пользователь нажимает "👥 Рефералы"
2. Получает ссылку: `https://t.me/botname?start=123456`
3. Отправляет друзьям
4. Когда друг активирует бесплатный тариф — реферер получает +3 дня

### Настройка бонуса:
```ini
REFERRAL_BONUS_DAYS=3  # Дней за каждого реферала
```

## 11. Решение проблем

### Бот не проверяет подписку
- Проверьте `CHANNEL_ID` в `.env`
- Убедитесь что бот — администратор канала
- Перезапустите бота

### Ошибка "no such column"
- Удалите `vpn_bot.db` (сделайте бэкап!)
- Запустите бота заново

### Пользователь не получает доступ
- Проверьте логи: `grep "ERROR" logs/app.log`
- Убедитесь что 3x-ui доступен
- Проверьте `XUI_INBOUND_ID`

## 14. Следующие шаги

### Для запуска сайта:
1. Установите nginx
2. Настройте reverse proxy на порт 8080
3. Получите SSL сертификат (Let's Encrypt)

### Для мобильного приложения:
1. Используйте Flutter или React Native
2. Интегрируйте API бота
3. Добавьте встроенные покупки

### Для масштабирования:
1. Перейдите на PostgreSQL
2. Добавьте Redis для кэша
3. Используйте Docker для развёртывания
